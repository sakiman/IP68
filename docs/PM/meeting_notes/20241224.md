   A. 內網 Wi-Fi Channel - **優先**
   - 成本最低
   - 開發較單純

   B. 手持設備 VPN - **優先**
   - 同筆電 VPN，可於任何地點作業

   C. 反向代理 Reverse Proxy (API) - **再議**
   - 外部 Server 或 IT 設置的對外 API Server 皆可
   - 基本不進行 Business Logic 邏輯於此伺服
   - 此伺服只處理轉發要求

   D. 中轉池 (API + Database + FTP Server) - **此方案預設排除**
   - 成本最高
   - 部份運算可能會於中轉上執行
